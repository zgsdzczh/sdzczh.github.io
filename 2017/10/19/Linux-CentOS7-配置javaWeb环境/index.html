Linux CentOS7 配置javaWeb环境

发表于 2017-10-19

前言

CentOS（Community Enterprise Operating System，中文意思是：社区企业操作系统）是Linux发行版之一，它是来自于Red Hat Enterprise Linux依照开放源代码规定释出的源代码所编译而成。由于出自同样的源代码，因此有些要求高度稳定性的服务器以CentOS替代商业版的Red HatEnterprise Linux使用。本文将详细介绍CentOS云服务器上面的javaweb运行环境搭建

安装JDK

首先java -version查看是否有jdk

没有的话去
这里下一个，这里使用的是.tar.gz的安装包 ，使用FileZilla将其放入选中的文件夹下。
tar-xvf jdk-8u121-linux-x64.tar.gz 解压
接下来要配置环境变量：

  • vi /etc/profile
  • export JAVA_HOME=/usr/java/jdk/jdk1.8.0_121
  • export JRE_HOME=/usr/java/jdk/jdk1.8.0_121/jre
  • export CLASSPATH=.:$JAVA_HOME/lib$:JRE_HOME/lib:$CLASSPATH
  • export PATH=$JAVA_HOME/bin:$JRE_HOME/bin/$JAVA_HOME:$PATH
  • 保存 source/etc/profile
  • 执行 java -version

安装mysql

下载 wget http://repo.mysql.com/mysql-community-release-el7-5.noarch.rpm
安装 sudo rpm -ivh mysql-community-release-el7-5.noarch.rpm
sudo yum install mysql-server
重启service
mysqld restart
重置密码

mysql
 -u root

mysql use mysql;

mysql update user set password=password(‘123456‘) where user=‘root‘;

mysql exit;

mysql> source /home/admin.sql
